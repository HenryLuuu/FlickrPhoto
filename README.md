# FlickrPhoto

功能:
● 輸入搜尋字串
● 透過 API 取得搜尋結果

第一頁 (搜尋輸入頁)
● 有兩個輸入框
● 不可點擊的 button 和可點擊的 button 用不同顏色區別
● 點擊button會轉跳(push)到第二頁(搜尋結果頁)
第二頁 (搜尋結果頁)
● 顯示Flickr回傳的API結果,並且固定一行兩個 cell
● 此頁可下拉重新更新
● 此列表可上拉看到下一頁內容(可無限上拉)
● Cell 會顯示圖片(photo),與標題,Cell為正方形
我的最愛頁面
● tabbar 多一個分頁
● 在搜尋結果頁的每一個 cell 加上加入收藏的按鈕
● 當按加入收藏,到我的最愛頁面,可以看到該 photo 被加入
● 重開 APP 依然可以看到被加入的收藏
